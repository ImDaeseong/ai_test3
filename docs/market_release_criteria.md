# Market and Release Criteria

A&R/시장성 관점의 평가는 흥행을 보장하는 것이 아니라, 곡 공개 우선순위와 수정 방향을 정하기 위한 비교 기준이다.

## Core Axes

- Hook: 기억나는 멜로디, 문장, 사운드 사인이 있는가.
- Mood: 감정과 분위기가 분명한가.
- Visual Potential: 이미지, 뮤직비디오, 쇼츠 장면으로 확장하기 쉬운가.
- Channel Fit: `justaimusickr` 채널 방향과 맞는가.
- Personal Taste: 사용자가 계속 밀고 싶은 곡인가.
- Short-form Fit: 10-20초로 잘라도 작동하는 구간이 있는가.
- Playlist Fit: 특정 분위기/상황 플레이리스트에 들어가기 쉬운가.
- Repeat Listen: 반복 청취 피로가 낮은가.

## Evidence Fields

- chorus strength
- first 3 seconds impact
- duration
- BPM and mood fit
- section energy curve
- lyrics/hook if available
- visual keywords if available
- user target platform

## Release Decision Labels

- Release First: 지금 공개 후보로 가장 강함.
- Revise Then Release: 수정 후 공개 가치 있음.
- Keep as Catalog: 지금은 보관, 채널/시즌/콘셉트 맞을 때 사용.
- Do Not Prioritize: 공개 우선순위 낮음.
- HOLD: 권리/안전/품질 문제로 공개 판단 보류.

## A/B Comparison Criteria

When comparing tracks:

1. Which hook is remembered after one listen?
2. Which has stronger visual or story packaging?
3. Which better fits the current channel direction?
4. Which needs fewer fixes before release?
5. Which has a clearer short-form cut?

## PASS / REVISE / HOLD

- PASS: 공개 목적과 플랫폼이 맞고, 훅/분위기/영상화 가능성이 충분하다.
- REVISE: 곡은 가능성이 있으나 훅, 길이, 후렴, 믹스, 영상 콘셉트 중 일부 보강 필요.
- HOLD: 저작권/복제/민감정보/외부 업로드 동의 문제 또는 사용자 채널 방향과 심각한 충돌.
