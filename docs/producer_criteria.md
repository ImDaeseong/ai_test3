# Producer Criteria

프로듀서 관점의 평가는 곡의 사운드 선택, 장르 해석, 리듬/악기/보컬 배치, 섹션별 에너지 설계를 본다.

## Core Questions

- 장르 라벨이 아니라 실제 리듬, 질감, 악기, 공간감으로 정체성이 드러나는가.
- 드럼과 베이스가 곡의 중심 감정에 맞는가.
- 보컬 톤과 공간감이 곡의 화자/장르와 맞는가.
- 섹션별로 레이어가 추가/제거되며 에너지가 움직이는가.
- 후렴, Drop, Bridge, Final Chorus에 제작적 보상이 있는가.
- AI 음악일 경우 평균적 장르 결과로 수렴하지 않고 구체적 질감이 있는가.

## Evidence Fields

- BPM and tempo range
- section energy curve
- frequency balance by band
- prompt style tags if available
- instrument tags if available
- vocal style tags if available
- chorus-to-verse energy delta

## Scoring Rubric

| Item | Weight | Good Signal | Weak Signal |
|---|---:|---|---|
| Sound identity | 20 | 리듬/질감/공간감이 구체적 | 장르명만 있음 |
| Groove and low-end design | 20 | 킥/베이스 역할 분리 | 저역 충돌/힘 없음 |
| Vocal production fit | 15 | 보컬 톤과 공간이 곡과 맞음 | 보컬이 떠 있거나 묻힘 |
| Arrangement arc | 20 | 섹션별 레이어 변화 뚜렷 | 처음부터 끝까지 평평함 |
| Hook production | 15 | 후렴에서 사운드 보상 있음 | 후렴이 커지지 않음 |
| Originality within genre | 10 | 익숙함과 새로움 균형 | 평균적인 템플릿 느낌 |

## PASS / REVISE / HOLD

- PASS: 곡의 정체성이 사운드로 설명되고, 후렴/브리지/마지막 후렴에 제작적 변화가 있다.
- REVISE: 악기 배치, 저역 설계, 보컬 공간, 섹션별 레이어 중 1-3개 개선 필요.
- HOLD: 실존 프로듀서/아티스트의 사운드 복제를 직접 요구하거나, 특정인의 고유 스타일 모방을 목표로 한다.
