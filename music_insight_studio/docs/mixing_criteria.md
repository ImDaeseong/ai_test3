# Mixing Criteria

믹싱 관점의 평가는 주파수 밸런스, 보컬/중심 악기 명료도, 저역 충돌, 공간감, 다이내믹을 본다.

## Measurable Proxies

- sub bass: 20-60 Hz
- bass: 60-250 Hz
- low-mid: 250-500 Hz
- mid: 500 Hz-2 kHz
- upper-mid: 2-4 kHz
- presence: 4-8 kHz
- brilliance: 8-16 kHz
- RMS mean/std
- dynamic range proxy
- spectral centroid/rolloff
- stereo width if available later

## Core Questions

- 저음이 충분하지만 탁하거나 과하지 않은가.
- 킥과 베이스가 같은 공간을 과하게 차지하지 않는가.
- 보컬 또는 중심 멜로디가 1-4 kHz 부근에서 묻히지 않는가.
- 4-8 kHz 존재감이 훅을 살리되 피곤하지 않은가.
- 8-16 kHz가 답답하지 않지만 치찰/쏨이 과하지 않은가.
- 리버브/공간감이 감정에 맞고 중심을 흐리지 않는가.
- 벌스와 후렴의 에너지 차이가 실제로 느껴지는가.

## Report Categories

- Low-end balance
- Vocal/melody clarity
- Midrange mud
- High-end fatigue
- Space and depth
- Dynamic contrast
- Mix translation risk for phone speakers and earbuds

## PASS / REVISE / HOLD

- PASS: 주요 대역이 균형 있고, 중심 요소가 선명하며, 반복 청취 피로가 낮다.
- REVISE: 저역, 중역 혼탁, 고역 피로, 보컬 밸런스, 공간감 중 일부 조정 필요.
- HOLD: 오디오 품질이 분석 불가능할 정도로 깨졌거나, 파일 손상/무음/극단적 클리핑으로 평가 신뢰도가 낮다.

## Uncertainty Note

보컬 밸런스와 악기별 충돌은 stem 분리 없이 추정이다. 리포트에는 `stem analysis not available`을 명시한다.
