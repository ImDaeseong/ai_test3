# Copyright and Content Policy

## Purpose

Music Insight Studio analyzes user-uploaded music. Commercial use requires explicit copyright and content boundaries.

## User responsibility

Users must confirm they have the right to upload and analyze the audio.

Allowed:

- User-created songs.
- Licensed works the user has permission to analyze.
- AI-generated tracks the user is allowed to use.
- Public-domain or properly licensed materials.

Not allowed:

- Uploading copyrighted songs without permission.
- Using the tool to clone a living artist's voice.
- Asking for exact artist impersonation.
- Using reference tracks in a way that violates rights.

## Product behavior

The product may:

- Analyze broad musical traits.
- Suggest mix/master/arrangement improvements.
- Suggest high-level style tags.
- Explain market/release fit as an estimate.

The product must not:

- Promise revenue or platform performance.
- Claim professional mastering certification.
- Generate instructions to impersonate a living artist.
- Store copyrighted reference audio without explicit user rights and retention policy.

## Reference-track comparison

Reference comparison is a later feature and requires:

- user rights confirmation,
- no raw reference sharing,
- no exact-clone recommendation,
- clear comparison limits.

## Takedown process

Before public launch, define:

- contact channel,
- takedown review process,
- user notification process,
- repeat-infringer policy if accounts exist.
