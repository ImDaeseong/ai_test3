# Marketability Scoring

## Scores

Each score should be 0-100 and include a short reason.

- YouTube reaction potential
- Spotify/Apple Music playlist fit
- Melon/Korean audience fit
- Shorts/Reels/TikTok usability
- repeat-listen potential
- chorus hook strength
- release readiness

## Inputs to Scoring

- BPM range
- duration
- dynamic contrast
- energy curve
- chorus section strength
- vocal/presence proxy
- genre and mood tags
- lyric/prompt structure when available

## Rule Style

Scores must be deterministic in MVP. If an LLM later rewrites explanations, the numeric score should still come from documented rules unless explicitly changed.
