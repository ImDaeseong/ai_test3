# AI Music Analysis Scope

## Evaluation Areas

- AI-like repetition
- vocal naturalness
- pronunciation and lyric delivery
- section transition smoothness
- chorus memorability
- arrangement variation
- emotional consistency

## Inputs

- audio file
- optional Suno prompt
- optional lyrics
- optional target genre and platform

## MVP Method

MVP uses rule-based checklists and measurable audio proxies. Later versions may add LLM-based review or specialized classifiers with explicit opt-in.

## Output Language

Reports should be practical Korean, focused on release decisions and production improvements.
